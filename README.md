# CMTG Deploy-ready Dark Package

Files:
- index.html
- api/transcript.js
- vercel.json

How to deploy:
1. Replace your repository root with these files.
2. Keep `index.html` at repo root.
3. Keep `api/transcript.js` under `/api`.
4. Push to GitHub.
5. Vercel will redeploy automatically.

Note:
- Current holder / peer / CECL values include static placeholders.
- Transcript is served through a Vercel Function stub.
